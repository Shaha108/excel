Project Report:
This project "Jump & Solve" was developed by SHAKHZOD & JONY. The game is inspired by Subway Surfers, featuring dynamic movements, obstacles, and logical challenges. The project was divided as follows:

- SHAKHZOD: Game mechanics, character movement, and level design.
- JONY: Puzzle system, UI/UX design, and documentation.

The objective is to jump over obstacles and solve puzzles to proceed. As levels increase, challenges become harder. This project helped us understand JavaScript game development and problem-solving techniques.
